# Therapist Website Redesign

A UX + Frontend case study where I redesigned a therapist booking website to improve trust, accessibility and booking conversion.

## Problem

Users felt anxious and unsure while booking therapy sessions. The original website lacked emotional comfort, clarity and clear CTAs.

## Goal

Create a calm, trustworthy and accessible experience that helps users confidently book a session.

## My Role

UX Research • UX Design • UI Design • Frontend Development

## Features

* Therapist profile pages
* Session booking flow
* Accessible typography & contrast
* Mobile-first responsive layout
* Trust-building visual design

## Tech Stack

Next.js
TypeScript
TailwindCSS

## Live Preview

https://therapist-website-redesign-olbn62ron-abhikawles-projects.vercel.app

## What I Learned

Designing for mental-health products requires reducing cognitive load and increasing emotional reassurance, not just usability.
